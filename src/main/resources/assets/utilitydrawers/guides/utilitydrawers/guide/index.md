# Utility Drawers

Utility Drawers adds simple, bulk item and fluid storage blocks that pack far more capacity into a single block than a chest or tank ever could. This guide covers everything from basic drawers to wireless networking.

Hold **G** (default keybind) while looking at any Utility Drawers item to jump straight to its page.
